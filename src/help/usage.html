使い方（Usage）

チュートリアル（※執筆中）

曲つくりの簡単な解説です。

ＭＭＬコンパイラ（nsc.exe）

ＭＭＬコンパイラー"nsc.exe"は、曲・効果音を作成するためのWindows用32bitアプリケーションです。
MMLで記述されたBGM, SEを、本音源ドライバーで再生可能なデータに変換します。
アセンブリ言語への出力のほか、NSF music formatへの出力に対応しています。

This is MML compiler.
It can convert the MML(Music Macro Langage) to the deta for this sound driver.
An output formats are as followss.
* Source code of the assembly language.
* NSF music format.

音源ドライバー（nsd.lib）

音源ドライバー"nsd.lib"は、曲・効果音を制御するためのファミリーコンピューター用のデバイスドライバです。
開発環境cc65用のオブジェクト・ライブラリとして開発しています。

This is sound driver to control BGM and SE for NES.
It have been developed as object library for cc65.
