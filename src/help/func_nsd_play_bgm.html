NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ 音源ドライバー＼ 関数リファレンス＼

nsd_play_bgm()

定義（Define）

        void    __fastcall__    nsd_play_bgm(void* bgm );

引数（Inputs）

void* bgm … 曲(BGM)データのポインターを指定してください。
Please set the pointer of music data.

戻り値（Outputs）

無し（no output）

内容（Contents）

引数void* bgmで指定されたポインターの曲(BGM)データの再生を開始します。

Sound driver will start the playing music that was set to void* bgm.

注意（Notes）

内部で、nsd_stop_bgm()関数を呼んでいますので、再生停止は必要ありません。
また、曲(BGM)データ以外のポインターを指定しないで下さい。暴走する可能性があります。

本関数を呼ぶ前に、ＲＯＭは曲データ及び効果音データがあるバンクに切り替えて置いてください。

Do not need to stop the music because function `nsd_stop_bgm()' are called by this function.
And, MUST NOT set the other pointer to the void* bgm. If set the other pointer, sound driver may 暴走.
