MML reference: Detune

定義（Define）

        Ddetune1
        D%detune2

引数（Inputs）

detune1

cent単位のディチューンです。
-128〜127の範囲で指定してください。
16が100[cent]（半音）に相当します。（Default: detune1 = 0）

detune2

レジスター単位のディチューンです。
-128〜127の範囲で指定してください。（Default: detune2 = 0）

内容（Contents）

ディチューンを指定します。

注意（Notes）

例（Sample）

    D%-1     /* register detune = -1 */
