MML reference: Relative Volume

定義（Define）

        ((volume)
        )(volume)

引数（Inputs）

volume

引数を書いた場合は、指定の値だけ音量を相対変化させます。
省略した場合のデフォルトは、1です。

内容（Contents）

(
音量を下げます。
)
音量を上げます。

注意（Notes）

引数を書いた場合は、その数量だけ相対音量コマンドを出力します。
リピート中などでない場合は、音量コマンドで書いた方がシーケンスのサイズは小さくなります。

2A03の三角波、⊿PCMトラックでは音量指定はできません。

例（Sample）

    [4 c& ) ]     /* cresc. */
    [4 c& ( ]     /* decresc. */
