MML reference: 早送り

定義（Define）

        jn

引数（Inputs）

n

0
早送りを無効にします。
1
早送りを有効にします。

内容（Contents）

曲を j1 の地点から、 j0 の地点へ早送りすることが可能になります。

注意（Notes）

ドライバー側実装の機能です。
早送り期間が長くフレームオーバーが多発した場合、暴走する可能性がある点、注意してください。

例（Sample）

bgm(0){

    TR1         j1          //ここから、早送りします。

    t120

    TR1 l4  o5  c1<
    TR2 l4  o4  e1
    TR3 l4  o3  c1  j0      //ここまで、早送りします。。

    TR1     b1	
    TR2     d1	
    TR3     g1	

}
